Popover component for displaying floating content triggered by user interactions.

The Popover component provides a way to display additional content in a floating panel that appears when triggered by a user action. It's built on top of Radix UI primitives and provides smooth animations, proper positioning, and accessibility features.

## Features
- **Accessible**: Built with proper ARIA attributes and keyboard navigation
- **Positioning**: Automatic positioning with collision detection
- **Animations**: Smooth enter/exit animations with CSS transitions
- **Composable**: Modular components for flexible layouts
- **Controlled**: Support for controlled and uncontrolled modes
- **Customizable**: Flexible styling and alignment options

## Usage
```tsx
import {
  Popover,
  PopoverTrigger,
  PopoverContent,
  PopoverAnchor,
} from '@repo/ui/components/Popover';

// Basic popover
<Popover>
  <PopoverTrigger asChild>
    <Button variant="outline">Open Popover</Button>
  </PopoverTrigger>
  <PopoverContent>
    <div className="space-y-2">
      <h4 className="font-medium">Popover Content</h4>
      <p className="text-sm text-muted-foreground">
        This is the content inside the popover.
      </p>
    </div>
  </PopoverContent>
</Popover>

// With icon trigger
<Popover>
  <PopoverTrigger asChild>
    <Button size="icon" variant="ghost" aria-label="Settings">
      <Settings className="size-4" />
    </Button>
  </PopoverTrigger>
  <PopoverContent>
    <div>Settings content</div>
  </PopoverContent>
</Popover>

// Controlled popover
const [open, setOpen] = useState(false);

<Popover open={open} onOpenChange={setOpen}>
  <PopoverTrigger asChild>
    <Button>Controlled Popover</Button>
  </PopoverTrigger>
  <PopoverContent>
    <div>Controlled content</div>
  </PopoverContent>
</Popover>
```
